# Python Web Scraping Portfolio

This repository contains Python scripts developed for automated web scraping and data extraction.  
The scripts demonstrate the ability to collect structured data from websites and export results to Excel or CSV.

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Excel Data Export

---

## Projects

### 1. Website Title Scraper
File: `scraper_titles.py`

Extracts article titles from a website.

Features:
- Sends HTTP request to webpage
- Parses HTML content
- Extracts structured text data

---

### 2. Data Scraper to Excel
File: `scrape_to_excel.py`

Scrapes website data and exports it to an Excel file.

Features:
- Collects titles and links
- Stores data using Pandas
- Saves structured output to `.xlsx`

---

### 3. Multi-Page Web Scraper
File: `multipage_scraper.py`

Extracts data from multiple pages automatically.

Features:
- Iterates through paginated pages
- Extracts repeated content
- Handles large datasets

---

### 4. Business Lead Scraper
File: `business_lead_scraper.py`

Collects website leads and exports them into an Excel spreadsheet.

Features:
- Extracts structured business data
- Organizes leads in tabular format
- Exports to Excel for easy analysis

---

## Example Output

The scripts generate structured datasets such as:
